Abstract

This project applies unsupervised statistical learning techniques to the UCI Wine dataset to explore whether wines can be grouped based solely on their chemical characteristics. Principal Component Analysis (PCA) is used for dimensionality reduction and visualization, while K-means and hierarchical clustering are applied to identify natural groupings in the data. Model selection is performed using the elbow method, and clustering results are evaluated by comparing discovered clusters with known wine cultivars using confusion tables and the Adjusted Rand Index. The results show that unsupervised methods are able to recover the underlying structure in the data, revealing clear groupings that correspond closely to the three wine cultivars.

Introduction

Statistical learning methods are widely used to discover structure and patterns in multivariate data. In many real-world applications, labeled responses are unavailable, making unsupervised learning techniques essential for exploratory analysis. These methods allow researchers to uncover hidden groupings, reduce dimensionality, and understand relationships among variables without prior knowledge of class labels.

In this project, unsupervised learning techniques are applied to a real-world chemical analysis dataset of wines from the same region in Italy. The wines are derived from three distinct grape cultivars, and each sample is described by thirteen chemical measurements (such as alcohol content, malic acid, color intensity, and proline). Although the true wine types are known in the dataset, they are treated as unknown during model training and only used later for evaluation.

Research question:

Can unsupervised statistical learning methods identify natural groupings of wines based solely on their chemical characteristics, and how well do these groupings correspond to the known wine cultivars?

To answer this question, we will use Principal Component Analysis (PCA) for dimensionality reduction and visualization, followed by K-means clustering and hierarchical clustering to identify and compare wine groups.

Data Description

The dataset used in this project is the UCI Wine dataset, which contains the results of chemical analyses performed on wines grown in the same region of Italy but derived from three different grape cultivars. After loading and assigning descriptive column names, the dataset consists of 178 observations and 14 variables, where:

1 variable (Type) represents the true wine cultivar (1, 2, or 3),

13 variables represent chemical measurements such as alcohol content, acidity levels, phenolic compounds, color intensity, and proline concentration.

The structure of the dataset (str) confirms that the chemical attributes are numeric, while the cultivar label is stored as an integer. A summary of the chemical variables indicates substantial variation across wines. For example:

The distribution of wine types is slightly imbalanced, with 59 samples of Type 1, 71 samples of Type 2, and 48 samples of Type 3, as shown by the frequency table.

A missing-value check confirms that there are no missing entries in any variable (colSums(is.na(wine)) = 0 for all columns). This ensures that the dataset can be used directly for PCA and clustering without the need for imputation or data cleaning.

Before applying PCA and clustering, the chemical variables were examined and found to be on very different numerical scales. For example, Proline values are in the hundreds, while several phenolic measurements are below 1. Because PCA and distance-based clustering are both sensitive to variable scale, features with larger ranges would dominate the analysis and distort the results. To prevent this, all variables were standardized using scale() so that each has mean 0 and standard deviation 1. The verification step confirms that the scaled data meet these conditions, ensuring that all chemical features contribute equally to PCA and clustering.

Discussion of Insights

Key Findings from Unsupervised Learning

The core finding is the remarkable effectiveness of unsupervised statistical learning in recovering the inherent structure of the wine data. By using only the 13 chemical features, both K-means and Hierarchical Clustering successfully identified three distinct groups, which aligns perfectly with the known number of wine cultivars.

Principal Component Analysis (PCA) Insights

The PCA was a critical initial step, reducing the 13-dimensional data into a two-dimensional space that retained 55.4% of the total variance. The PCA scatter plot visually confirmed the underlying three-group structure, demonstrating that the chemical composition alone is sufficient to separate the cultivars.

PC1 (36.2% variance) appears to primarily differentiate Type 1 and Type 3 wines from Type 2 wines, suggesting a strong chemical contrast based on factors like Proline and Alcohol (positively correlated with PC1), and Flavanoids and OD280_OD315 (negatively correlated with PC1, see biplot).

PC2 (19.2% variance) further separates Type 1 from Type 3.

Clustering Performance and Feature Influence

The confusion tables confirmed that both clustering models closely match the true cultivars. Hierarchical clustering demonstrated a slightly cleaner separation, especially between Types 1 and 2, resulting in fewer overall misclassifications than K-means.

The necessity of data standardization was evident: the large range of values for variables like Proline (ranging from 300$ to 1700) would have improperly dominated the distance metrics used in clustering had the data not been scaled. The success of the clustering methods confirms that, once standardized, all chemical features contribute equitably to defining the groups.

Conclusion and Limitations

Conclusion

The research question, “Can unsupervised statistical learning methods identify natural groupings of wines based solely on their chemical characteristics, and how well do these groupings correspond to the known wine cultivars?” is answered with a strong affirmative. Unsupervised methods, specifically K-means and Hierarchical Clustering applied to standardized chemical data, successfully identified three distinct clusters that align closely with the three true wine cultivars (Types 1, 2, and 3). The high degree of alignment, quantified by the confusion matrices, demonstrates that the underlying chemical composition is the dominant factor defining these wine groups.

Limitations

Dependency on Data Scale: The success of the clustering was highly dependent on the initial standardization of the data. Without scaling, features with large ranges (like Proline) would have heavily skewed the results, a limitation inherent to distance-based clustering.

Model Selection: The determination of the optimal number of clusters, k=3, relied on the Elbow Method, which can sometimes be ambiguous. While the result aligned with the known answer, in a truly unsupervised scenario, alternative metrics would be necessary for a more robust k selection.

Generalizability: The analysis is restricted to the UCI Wine dataset, which consists of wines from a single region in Italy. Applying these models to wines from different regions or with different chemical measurement techniques may require recalibration.
